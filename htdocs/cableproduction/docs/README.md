# CableProduction - Агрегатор производителей кабельной продукции

## Описание проекта
Веб-приложение для сбора и представления информации о российских производителях кабельной продукции с удобным поиском и фильтрацией.

## Функциональность
- Поиск производителей по названию, городу, типу кабеля
- Фильтрация по городу, типу кабеля, наличию цены
- Карточки производителей с контактами и описанием
- Детальные страницы производителей с каталогом продукции
- Админ-панель для управления данными (CRUD)

## Технологический стек
- **Frontend**: HTML5, CSS3, JavaScript (ES6+), Bootstrap 5
- **Backend**: PHP 7.4+, MySQL 8.0
- **Сервер**: XAMPP (Apache, MySQL)
- **API**: RESTful JSON API

## Установка и запуск

### 1. Требования
- XAMPP (Apache, MySQL, PHP)
- Браузер с поддержкой ES6+

### 2. Установка
1. Скопируйте проект в `C:\xampp\htdocs\cableproduction`
2. Запустите XAMPP Control Panel
3. Запустите Apache и MySQL

### 3. Запуск
1. Откройте в браузере: `http://localhost/cableproduction/`
2. Для админ-панели: `http://localhost/cableproduction/admin_login.html`
- Логин: `admin`
- Пароль: `admin`

## API Endpoints

### Производители
- `GET /backend/api/manufacturers.php` - список всех производителей
- `GET /backend/api/manufacturers.php?id=1` - производитель по ID
- `POST /backend/api/manufacturers.php` - создание производителя
- `PUT /backend/api/manufacturers.php` - обновление производителя
- `DELETE /backend/api/manufacturers.php` - удаление производителя

### Продукция
- `GET /backend/api/products.php` - список всей продукции
- `GET /backend/api/products.php?manufacturer_id=1` - продукция производителя
- `GET /backend/api/products.php?type=Силовой` - фильтр по типу
- `POST /backend/api/products.php` - создание продукта
- `PUT /backend/api/products.php` - обновление продукта
- `DELETE /backend/api/products.php` - удаление продукта
